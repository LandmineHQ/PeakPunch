# BuddyClimb

Use `E` to make your teammates carry you, even when they have a backpack.

If climbing feels too tiring, let a teammate carry you up the mountain before you conquer it yourself.

Optional config: enable `Backpack.EnableBackpackTransfer` to climb onto backpack-wearing teammates by moving their backpack onto you first.

Config changes are hot-reloaded while the game is running. Edit the BuddyClimb config file in BepInEx and the new values apply automatically without restarting PEAK.
